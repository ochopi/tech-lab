Si se descarga la iso de Bazzite Deck Version, esta arrancará directamente en Gamescoop mode cargando Steam Primero en lugar de lanzar el escritorio KDE o GNOME.


# METODO 1: EMPTY FILE:

Based on bazzite-autologin service script https://github.com/ublue-os/bazzite/blob/14cb28a/system_files/deck/shared/usr/libexec/bazzite-autologin
You can place an empty /etc/bazzite/desktop_autologin file to boot to desktop mode by default
This should work for both KDE and GNOME version.
Make sure bazzite-autologin service is enabled: systemctl enable bazzite-autologin

'''
sudo touch /etc/bazzite/desktop_autologin
'''


To go back to default after doing this, just delete the empty bazzite-autologin file:
'''
sudo rm /etc/bazzite/desktop_autologin
'''



# METODO 2: UJUST:

Si se desea que se inicie directamente en el escritorio se puede usar un comando ujust oculto para forzar este comportamiento

ADVERTENCIA:
Este comando esta oculto porque es sabido que causa inestabilidad en el sistema.
Uselo bajo su propio riesgo.
Para revertir el funcionamiento basta con volver a ejecutar el mismo comando

'''
ujust _toggle-autologin
'''

NOTA:
Segun el mismo archivo Return.desktop
al estar en modo Desktop KDE se puede cambiar al modo Gaming con:
systemctl start return-to-gamemode.service


Return.desktop:
-----------------------
#!/usr/bin/env xdg-open
[Desktop Entry]
Name=Return to Gaming Mode
Exec=systemctl start return-to-gamemode.service
Icon=steamdeck-gaming-return
Terminal=false
Type=Application
StartupNotify=false
-----------------------
