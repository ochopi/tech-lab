# INSTALACION Y CONFIGURACION DE ONEDRIVE EN LINUX-BAZZITE

## Descargar e Instalar el motor
sudo rpm-ostree install onedrive

## Reiniciar
systemctl reboot

## INICIAR EL PROGRAMA
Se puede usar directamente en la linea de comandos simplemente con "onedrive" o Descargar una GUI
OJO: Elegir solo una. o CLI o GUI


## CONFIGURACION CLI
### CLI AUTENTICACION
NOTA: SI PLANEA USAR LA GUI, SALTE HASTA LA SECCION GUI EN ESTE MOMENTO.
ejecutar el comando
'''
onedrive
'''
- Se nos dara una URL. Hacer clic o copiarla al navegador
- Hacer la autenticacion en el Navegador
- Al terminar la autenticacion, copiar (rapidamente) la URL de redireccion
- Pegar esa URL en la terminal.
- Listo ya estamos autenticados

### CLI SINCRONIZACION
'''
onedrive --sync
'''
- Eso crea una nueva carpeta en ~/OneDrive y comienza a descargar todo el contenido de tu OneDrive
- Las configuraciones de este perfil estan almacenadas en ~/.config/OneDrive

### CLI INICIO AUTOMATICO
Para que el motor de onedrive arranque con el sistema como servicio debemos hacer una unica vez el comando:
'''
systemctl --user enable onedrive
systemctl --user start onedrive
'''

## CONFIGURACION GUI
Si decides usar la GUI no debes configurar nada por linea de comando.
Descarga el .AppImage del repo oficial
https://github.com/bpozdena/OneDriveGUI

La GUI nos pedira crear un perfil la primera vez (o cargarlo, si ya teniamos uno)
En el perfil asignamos un NOMBRE y un punto de montaje. por lo general ~/OneDrive_NOMBRE

### GUI AUTENTICACION:
Se nos dara un link para autenticarnos en el navegador.
Luego de autenticar, el navegador nos redirigira a una URL (copiar esa url rapidamente)
Pegar esa url en el cuadro de texto.

### GUI SINCRONIZACION
Una vez autenticados, presionar el boton de SINCRONIZACION y eso comenzara la descarga
de todos tus archivos a la carpeta designada (por lo general ~/OneDrive_NOMBRE)

### GUI INICIO AUTOMATICO
- En la pestaña PROFILE marcar la casilla de "AutoSync on GUI Startup"
- En la ruedita de configuracion podemos marcar la casilla de
    * Iniciar OneDriveGUI automaticamente en el login
    * Iniciar OneDriveGUI minimizado
  Marcar ambas casillas para una experiencia windows-like


## EXTRAS
### CONFIGURAR SYSTEMCTL
'''
systemctl --user edit onedrive
'''

### EDITAR EL ARCHIVO:



'''
### Anything between here and the comment below will become the contents of the drop-in file

[Service]
ExecStart=/usr/bin/onedrive --confdir=/home/USUARIO/.config/onedrive/accounts/NOMBRE --monitor


### Edits below this comment will be discarded
'''




### GUARDAR LOS CAMBION CON ESC :wq ENTER
'''
systemctl --user daemon-reload
systemctl --user restart onedrive
'''
