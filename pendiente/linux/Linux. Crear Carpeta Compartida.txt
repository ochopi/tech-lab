sudo apt update && apt upgrade
sudo apt install samba

> (opcional, crear un usuario solo para compartir archivos: sambauser)
adduser sambauser
> Crearle una contraseña. Los demas datos no son tan importantes.

> Loguearse con ese usuario:
su sambauser
> Crear el directorio que vamos a compartir dentro de ese usuario:
cd
pwd
mkdir compartida
chmod 777 compartida/
ll

> configuracion:
vim /etc/samba/smb.conf

> Agregar al final:
[compartida]
path = /home/sambauser/compartida
browseable = yes
read only = no
valid users = sambauser

> Agregar el usuario sambauser a la lista de usuarios validos de Samba
> Y establecer una contraseña para acceder a los recursos compartidas
smbpasswd -a sambauser

> Crear un archivo dentro de la carpeta 
touch Archivo_Importante
exit

> Reiniciar el servicio (dependiendo de la distro):
service smbd restart
systemctl restart smdb
