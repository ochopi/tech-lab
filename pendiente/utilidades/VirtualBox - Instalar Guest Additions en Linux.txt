INSTALAR GUEST ADDITIONS EN LINUX

- Montar la imagen de Guest Additions, desde la barra de menu de la maquina virtual
  > Menu Dispositivos 
  > Insertar Imagen de CD de Guest Additions



- Verificar que esté montada correctamente
  > Abre una terminal y ejecuta:
lsblk

    Deberías ver algo como:
sr0    1024M  0  rom  /media/tu_usuario/VBox_GAs_7.0.XX


O bien revisa si puedes ver la carpeta desde el explorador:
📁 `/media/<tu_usuario>/VBox_GAs_...`



- Instala los paquetes necesarios
Antes de correr el instalador, necesitas herramientas de compilación y headers del kernel:
  > Abre una terminal y ejecuta:
sudo apt update
sudo apt install -y build-essential dkms linux-headers-$(uname -r)

    Este comando:
 ✅ Instala las herramientas para compilar código
 ✅ Instala DKMS para mantener los módulos actualizados
 ✅ Instala los headers del kernel actual
 ➡️ Resultado: tu sistema queda listo para compilar e instalar correctamente los módulos de VirtualBox Guest Additions 
    (video, carpetas compartidas, integración del mouse, etc).



- Ejecuta el instalador
Una vez hecho lo anterior, ve al directorio donde está montado el CD:
  > En una terminal, ejecuta:
cd /media/$(whoami)/VBox_GAs_*



  > Y ejecuta el instalador con privilegios de root:
sudo sh VBoxLinuxAdditions.run



- Reinicia la maquina virtual
Cuando termine (sin errores), reinicia tu máquina virtual:
  > En la terminal, ejecuta:
sudo reboot



- Verifica que funcionó
Después de reiniciar:
  > En la terminal, ejecuta:
lsmod | grep vbox


   Deberías ver algo como:
vboxguest
vboxsf
vboxvideo

    Y deberías notar:
* Mejor integración del mouse
* Resolución automática de pantalla
* Carpetas compartidas y portapapeles bidireccional funcionando.



- (Opcional) Problemas comunes

**🟥 Error: "Building the main Guest Additions module failed"**
👉 Asegúrate de que los headers coincidan exactamente con tu kernel:

uname -r
apt list --installed | grep headers


**🟥 No cambia resolución automáticamente**
👉 Instala también:
sudo apt install -y virtualbox-guest-x11

y reinicia otra vez.
